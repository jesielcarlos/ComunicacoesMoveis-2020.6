Calculadora 5G NR Throughput

Para 5G NR, a taxa de dados aproximada para um determinado número de portadoras agregadas em uma banda ou combinação de bandas é calculada usando a equação acima.
➤J: número de portadores de componentes agregados em uma banda ou combinação de bandas.
➤Rmax : 948/1024 .
• Vlayers(j) : É o número máximo de camadas.
➤Qm(j) : Ordem de modulação, Qm é 2 para QPSK, 4 para 16QAM, 6 para 32QAM, 8 para 256QAM.
➤f(j) : Fator de escala, pode assumir os valores 1/0.8/0.75/0.4 .
➤μ : 5G NR Numerologia, pode assumir os valores de 0 a 4.
➤Tsμ : Duração média do símbolo OFDM em um subframe para valor μ
•  Tsμ = 10-3/(14*2μ).
➤NPRBBW(j),μ : Maximum RB Alocação máxima de RB na largura de banda, BW(j)with com numerologia (μ), BW (j) é a largura de banda máxima suportada pela UE em determinada banda ou em combinações de banda. REs são agrupados em PRBs (blocos de recursos físicos). Cada PRB consiste em 12 subportadoras.
➤OH(j) : Overhead assume os seguintes valores
• [0.08] → Faixa de frequência FR1 para UL
• [0.14] → Faixa de frequência FR1 para DL
• [0.18] → Faixa de frequência FR2 para DL
• [0.10] → Faixa de frequência FR2 para DL
